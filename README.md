#  电影推荐系统

基于协同过滤的智能电影推荐系统，使用Django构建。

##  核心功能

-  个性化电影推荐
-  电影搜索与浏览  
-  用户评分系统
-  基于用户相似度的推荐

##  技术栈

### 前端
- Bootstrap 3

### 后端
- Django 5.2.8
- Django REST Framework

### 使用的推荐算法
- **算法类型**：基于用户的协同过滤
- **技术实现**：Surprise库的KNNBasic算法
- **相似度计算**：余弦相似度

### 数据存储
- SQLite3数据库
- MovieLens 100K数据集

## 算法原理
系统使用基于用户的协同过滤：
1. 计算用户间的余弦相似度
2. 找到最相似的K个邻居
3. 根据相似用户的喜好推荐电影

## 个人说明
对于前期的过程代码的实现我使用的是VS Code + Copilot Pro。对于前期来说无论是使用现成的数据集还是去爬取的电影数据，数据清洗是一个非常重要的步骤，需要去除空值、重复值、异常值。
这也是为了训练出来的模型能达到更好的预期。
对于一个普通学生来说，还是先使用现成的数据集来感受一下会更好上手；下一步我也会尝试去爬取实时数据来达到更好的效果。推荐算法的实现也有很多种，选取最契合数据的即可。在推荐系统的构建上，可以先
创建一个命令行脚本来实现一个简单的交互；如果想进一步提升项目的可用性，可以使用 Flask 或 Django 搭建一个简单的Web应用，我个人使用的是Django。
我个人来完成这个项目只是因为兴趣并且为了积累一定的经验，这也是一个很久之前的系统了，拿来上手还是挺不错的
